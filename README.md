Drosophila (fruit fly) population simulator. 🪰💻
Instead of simple random particles, it runs on a Finite State Machine (FSM) + Boids algorithm, mapping the exact life cycle: Egg → Larva → Pupa → Adult.
Fascinating to watch them:
🌡️ Hibernating near the core when <8°C
🔥 Dying from heat stress at >33°C
🎶 Performing courtship rituals and mating at ~25°C
Built entirely with HTML5 Canvas & JS. AI is getting crazy good at translating strict biological rules into code.
